# MULTIPLE OSCILLATOR VIRTUAL SYNTHESISER (M.O.V.S.)
JAYMIAN TURNER, 2024
===============

This Virtual Synthesiser was made for a school project.
Written in C++, with help from the JUCE library

!!JUCE 6 or above is required!! (JUCER file provided)
